Este projeto consiste na criação de um aplicativo para dispositivos móveis android cujo nome é “Biblioteca Pocket” e por ser para telemóveis, provém mais acessibilidade para a leitura de livros. A aplicação é basicamente uma biblioteca virtual e contém todo um catálogo de livros, sistema de favoritos, filtros de pesquisa, login etc. Este projeto também propõe um outro aplicativo que fica responsável pela parte de administrar os livros e usuários do aplicativo principal. Ambos são feitos usando a linguagem de programação Java e o banco de dados Firebase. Esta aplicação pode ser feita por programas de computador como Android Studio ou Eclipse e no Android ferramentas como AIDE ou Sketchware que também funcionam de forma parecida.
